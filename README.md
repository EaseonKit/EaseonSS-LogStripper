## Overview


**Log Stripper** is a lightweight Fabric mod that enhances log stripping in Minecraft. Hold an axe in one hand and a log in the other, then right-click to strip it instantly.

## Commands
All commands require OP level 2 permission.

| Command                         | Description          |
| ------------------------------- | -------------------- |
| /easeon logstripper            | View current status  |
| /easeon logstripper <on/off>   | Enable/disable       |

<br/>

## Configuration
```json
{
  "enabled": true,
  "requiredOpLevel": 2 // Requires a server restart to take effect.
}
```
`config/easeon/easeon.ss.logstripper.json`
